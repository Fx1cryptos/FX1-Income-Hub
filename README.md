# FX1 Income Hub

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)  
[![Website](https://img.shields.io/badge/Visit-FX1%20Income%20Hub-royalblue)](https://fx1hubs.short.gy)

FX1 Income Hub is a **professional Web3 earning portal**, curated by **FX1 Digital Hubs**.  
It is designed to help beginners and experienced users explore, learn, and earn across multiple blockchain ecosystems.  

---

## 🌐 Features

- **Multichain Earning Opportunities:** Ethereum, Base, Solana, Polygon, and more.  
- **Earning Categories:**  
  - Web3 Jobs  
  - Bounties  
  - Tasks to Earn  
  - Train AI To Earn  
  - Earn via Art & NFT  
  - DePin  
  - Cryptocurrency  
- **Card Layout:** 2 cards per row displaying platform favicon, title, description, and referral link.  
- **Wallet Integration:** Connect MetaMask, Coinbase Wallet, Phantom, and other wallets.  
- **Social & Portfolio Links:** Easily access FX1 Digital Hubs portfolio and social channels.  
- **Responsive Design:** Optimized for desktop, tablet, and mobile.  
- **AI Chatbot:** Beginner guidance, FAQs, and earning tips.  

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS  
- **Web3 Integration:** Thirdweb SDK, Ethers.js, Web3Modal  
- **Deployment:** Vercel / Builder.ai  
- **Design:** White background, Royal Blue accent color  
- **Version Control:** Git + GitHub
